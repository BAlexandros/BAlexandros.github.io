Steganographer is a command line program which hides one png image into another.

Description

Steganography is the practice of concealing information in plain sight, within non-secret data. Its advantage to cryptography is that encrypted information is very obvious to the prying eye because of its unintelligible nature, whereas data hidden using steganography methods are less likely to arouse suspicion.

This program allows the hiding of one image into another, as long as their file format is png.

Implementation

The program is written in C, and works as a command line tool. It uses libpng to read and write the png files. Once the files are successfully loaded, the least significant bits of the cover image are replaced by the most significant bits of the hidden image. The default number of bits is 4, which can be changed using the appropriate flag when calling the program.

The process can be reversed, decrypting an image, using the appropriate flag. Of course, this results in loss of information, since a number of least significant bits was lost during the process of hiding.

The tool was developed and works on Linux, and can be easily built with the help of the make utility.
